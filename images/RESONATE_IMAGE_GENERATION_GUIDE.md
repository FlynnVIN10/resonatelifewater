## QUICK START WORKFLOW

### Step 1: Choose AI Tool
- **Recommended:** DALL-E 3 (via ChatGPT Plus), Leonardo.ai, or Midjourney
- **Best for Nature Photography:** Midjourney (artistic quality) or DALL-E 3 (precise control)

### Step 2: Generate Images by Category
Follow the prompts in [AI Generation Prompts](#ai-generation-prompts) section below. Generate images in this order:
1. Hero Background (Waterfall/Nature) - **Priority 1**
2. Logo/Branding - **Priority 2**
3. Product Images - **Priority 3**
4. About Section Images - **Priority 4**
5. Lifestyle Images - **Priority 5**
6. Icons/Graphics - **Priority 6**

### Step 3: Optimize Images
- Compress to meet file size requirements (use TinyPNG, Squoosh, or ImageOptim)
- Convert to appropriate format (JPG for photos/backgrounds, PNG for logos, SVG for icons)
- Verify dimensions match specifications

### Step 4: Place in Directory Structure
- Follow the [Directory Structure](#directory-structure) section
- Use exact file names specified in prompts

## FILE NAMING CONVENTIONS

- Include category prefix when helpful
- Maintain consistency across all files

## OPTIMIZATION GUIDELINES

1. **Compression:**
   - Use TinyPNG, Squoosh, or ImageOptim
   - Target file sizes as specified per category
   - Maintain visual quality while reducing file size

2. **Format Selection:**
   - JPG for photos and backgrounds
   - PNG for logos (with transparency)
   - SVG for icons (vector graphics)

3. **Dimensions:**
   - Verify exact dimensions match specifications
   - Ensure aspect ratios are correct
   - Test responsive behavior
