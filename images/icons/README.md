# Icons Directory

**Purpose:** SVG icons for value propositions and benefits  
**Format:** SVG (vector)  
**Dimensions:** 256x256px viewBox  
**File Size:** <10KB each

## Required Files:
- `pure-water.svg` - Water droplet or clean water symbol
- `cost-savings.svg` - Dollar sign or savings symbol
- `eco-friendly.svg` - Leaf or recycling symbol
- `health.svg` - Heart or wellness symbol
- `installation.svg` - Tools or installation symbol
- `smart-tech.svg` - Technology or smart device symbol

## Usage:
These icons are used in value proposition cards and benefit sections. They should be modern, minimalist, and align with OMGEN WATER brand colors.

See `AI_IMAGE_GENERATION_GUIDE.md` for generation prompts.
