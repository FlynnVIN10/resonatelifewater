# Branding Directory

**Purpose:** OMGEN WATER brand logo in multiple formats  
**Format:** PNG (with transparency)  
**Dimensions:** Various (see files)  
**File Size:** <50KB each

## Required Files:
- `logo-full.png` - Full logo with wordmark (512x512px)
- `logo-icon.png` - Icon only (256x256px)
- `logo-horizontal.png` - Horizontal layout (1024x256px)

## Usage:
- `logo-full.png`: Used in footer, email signatures, marketing materials
- `logo-icon.png`: Used as favicon, app icon, social media profile
- `logo-horizontal.png`: Used in header/navigation, horizontal layouts

## Brand Guidelines:
- Primary colors: Blue (#007BFF), Black (#000000)
- Accent: Green (#28A745) for eco-friendly elements
- Style: Modern, clean, water-focused, professional
