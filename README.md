# Resonatelifewater

A modern, responsive landing page showcasing Resonatelifewater's innovative hydration technology.

## Features
- Premium hero storytelling with clear calls-to-action.
- Science-led brand narrative highlighting mineral-balanced hydration.
- Benefits grid, sustainability commitments, and social proof.
- Responsive navigation with smooth scrolling and interactive waitlist form.

## Technology Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Live Preview
Open `index.html` directly in your browser or serve locally:

```bash
python -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).

## Customization
- Update copy and media directly in `index.html`.
- Tweak the color palette, spacing, and effects in the `:root` variables at the top of `styles.css`.
- Adjust navigation behavior, scroll animations, or form handling within `script.js`.
